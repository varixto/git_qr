The jQuery Google Maps Plugin v.2, a lightweight (<4K) wrapper for Google Maps v.3, provides a simplified api for Google Maps JS API V3 for jQuery Mobile and jQuery UI Frameworks.

The AppLaud plugin for Eclipse by Mobile Developer Solutions enables rapid development of mobile web apps for Android, utilizing PhoneGap and its device APIs.
